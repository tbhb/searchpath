# Getting help

Whether you're just getting started or working on an advanced use case, these resources can help.

## Documentation

The [usage documentation](index.md) covers installation, core concepts, and common patterns. The [API reference](reference/api.md) provides detailed information on all public classes and functions.

## Discussions

For questions, ideas, or general discussion, use [GitHub Discussions](https://github.com/tbhb/searchpath/discussions). Your question and its answer help others in the community.

## Bug reports

Found a bug or have a feature request? Open an [issue on GitHub](https://github.com/tbhb/searchpath/issues). Please search existing issues first to avoid duplicates.
